Build The Community of Developers for Today by Following these Steps:

  • Focus on design architecture
  • Functional design vs perfect design
  • Support global community by sharing your expertise
  • Seek knowledge and develop learning muscles
  • Go after why to solve a project as opposed to what new technologies to use to complete the project
  • Give/receive feedback
  • Join Slack community for Developers
  • Build diverse projects using various technologies
  • Showcase Your Knowledge, skills and expertise in Portfolio
  • Be an active user of GitHub
